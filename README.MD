# EMFISIO

Sistema de Gestão de Atendimentos e Processos de Fisioterapia e Pilates.

## Sobre o projeto

O EMFISIO é um sistema full stack desenvolvido com o objetivo de gerenciar os principais processos de uma clínica de fisioterapia e pilates.

O sistema será desenvolvido como projeto de portfólio, aplicando conceitos de desenvolvimento backend, frontend, banco de dados, APIs REST, versionamento de código e metodologias ágeis.

## Objetivo

Criar uma aplicação capaz de gerenciar:

- Clientes
- Profissionais
- Serviços
- Agendamentos
- Pagamentos
- Comissões
- Pacotes
- Indicadores
- Relatórios

## Tecnologias previstas

- Java
- Spring Boot
- React
- SQL Server
- APIs REST
- Git
- Azure DevOps
- Postman

## Módulos previstos

### Clientes

Cadastro, edição, busca e histórico dos clientes atendidos pela clínica.

### Profissionais

Cadastro dos profissionais, especialidades, horários de atendimento e percentuais de comissão.

### Serviços

Cadastro dos serviços oferecidos, como fisioterapia, pilates, fisioterapia pélvica, osteopatia, avaliação inicial, pacotes e mensalidades.

### Agendamentos

Controle de agenda, horários disponíveis, cancelamentos, faltas, reagendamentos e bloqueios de agenda.

### Pagamentos

Registro de pagamentos, controle de pendências e confirmação de recebimentos.

### Comissões

Cálculo de comissões dos profissionais após confirmação do pagamento.

### Dashboard

Indicadores financeiros e operacionais, como faturamento, atendimentos realizados, faltas e cancelamentos.
