# Metodologia Ágil — EMFISIO

## Abordagem

O projeto EMFISIO será organizado utilizando práticas de Scrum e Kanban.

## Scrum

Será utilizada a ideia de sprints para dividir o desenvolvimento em ciclos curtos.

Cada sprint terá:

- Objetivo
- Lista de tarefas
- Entregas esperadas
- Revisão
- Retrospectiva

## Kanban

Será utilizado um quadro com as seguintes colunas:

- Backlog
- A Fazer
- Em Desenvolvimento
- Em Teste
- Concluído

## Sprint 1 — Fundação do Projeto

### Objetivo

Criar a estrutura inicial do projeto, documentação e versionamento.

### Tarefas

- Criar repositório EMFISIO
- Criar README.md
- Criar pasta docs
- Criar documento de requisitos
- Criar documento de regras de negócio
- Criar documento de user stories
- Criar documento de modelo inicial do banco
- Criar estrutura de pastas
- Realizar primeiro commit

### Entregas esperadas

- Estrutura inicial do projeto criada
- Documentação inicial preenchida
- Projeto versionado com Git
